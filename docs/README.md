## Convenciones de Nomenclatura

- **MAYÚSCULAS**: Documentos principales y oficiales
- **kebab-case**: Guías y documentación técnica
- **Prefijos por tipo**:
  - `PLAN_`: Documentos de planificación
  - `ANALISIS_`: Análisis técnicos
  - `FASE_`: Documentación de fases
  - `REPORTE_`: Reportes y validaciones

## Mantenimiento

Esta estructura debe mantenerse actualizada. Al agregar nueva documentación:
1. Clasifica el documento según su propósito
2. Colócalo en la carpeta apropiada
3. Actualiza este README si es necesario
4. Mantén las convenciones de nomenclatura
